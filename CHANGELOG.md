# 版本更新说明

## v1.1.0 - 2026-06-01

### 新增

- **随机分配模式**：新增 🎲 随机分配选项，纯随机打乱玩家分配阵营，适合娱乐局或测试
- **分配模式选择页**：导入玩家后新增一步，让用户选择「均衡分配」或「随机分配」
- 随机模式自动跳过游戏模式选择页（步骤指示器显示删除线标记）
- 结果页随机模式显示「随机分配」标签，隐藏均衡性告警
- **文字导出**：结果页「复制结果」按钮，格式化纯文本写入剪贴板，方便粘贴到 QQ/Discord
- **历史记录**：自动保存最近 5 次分队结果，独立历史页可浏览记录列表并点击查看详情，存储于 `%USERPROFILE%\Documents\BF6TeamBalancer\history.json`
- **操作页**：结果页之后新增操作汇总页，提供「重新导入」「重新分配」「退出」三个按钮

### 优化

- UI 步骤指示器从 4 步扩展为 6 步（导入 → 分配模式 → 游戏模式 → 自定义小队 → 结果 → 操作）
- 随机模式下返回上一步自动跳过游戏模式页
- 自定义小队绑定在随机模式下仍然生效（不能拆散）
- 「重新导入」可清空数据回到起点，无需重启程序
- 「重新分配」保留玩家数据，回到分配模式选择

### 算法

- 新增 `random_allocate()` 函数：random.shuffle 打乱 → AABB 轮流分配 → 4 人编队 → 候补抽取
- 均衡模式算法无变动，原有行为完全保留
